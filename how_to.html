About this guide

This guide was designed to inform the reader about JavaScript functions and how those functions can be used to create classes within your JavaScript code. Creating classes in JavaScript will allow for greater flexibility, maintainability and scalability in your project.

JavaScript Overview

JavaScript is a computer programming language used as a client side scripting language. JavaScript will interact with the user without the need to send and retrieve information to and from the server. JavaScript is very popular because it is very powerful and useful in web development. JavaScript can dynamically add and change html elements on a html page.

JavaScript Functions

Functions are a very useful tool in computer programming. They allow the programmer to encapsulate a task into a block of code that can be called multiple times throughout the program.

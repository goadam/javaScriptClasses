About this guide

This guide was designed to inform the reader about JavaScript functions and how those functions can be used to create classes within your JavaScript code. Creating classes in JavaScript will allow for greater flexibility, maintainability and scalability in your project. Technically JavaScript has no class key word and therefore has no classes. JavaScript is a prototyped-based language which differs from a class-based language. But it is possible to use JavaScript like a class-based language which I will show in this guide and therefore will be using the class keyword to describe certain topics.

JavaScript Overview

JavaScript is a computer programming language used as a client side scripting language. JavaScript will interact with the user without the need to send and retrieve information to and from the server. JavaScript is very popular because it is very powerful and useful in web development. JavaScript can dynamically add and change html elements on a html page.

JavaScript Functions

Functions are a very useful tool in computer programming. They allow the programmer to encapsulate a task into a block of code that can be called multiple times throughout the program.

JavaScript Classes

JavaScript is an object-oriented language. Which means JavaScript can be organized using object rather than a series of instructions and procedures. An object is a grouping of related data that can be similar to real world objects. Usually an object will have variables and methods which manipulate those variables.

Below is an example to create a basic class and object. You first want to create a function with its data members defined. Then create a variable and set it equal to an instance of the function you just created by using the keyword new. The "this" keyword refers to the instance of the object the function belongs to. Whatever object called the function is what "this" refers to. The variables "number1" and "number2" are initialised to the values shown but can be updated later on. 
   
	//Create the class
	function myFunction() {
		this.number1 = 5;
		this.number2 = 7;
		this.addNumbers = function() {
			return this.number1 + this.number2;
		}
	}
			
			
	//Create an instance of the class
	var classInstance = new myFunction();

JavaScript Prototype

			function car(name) {
				this.name = name;
				this.price = 0;
				this.set_price = function(price) {
					this.price = price;
				}
			}
A prototype is an object. Every object in JavaScript has a prototype and all objects inherit their properties from their prototype. This is a great feature JavaScript provides. You can use the prototype object to add member functions at runtime. In the example above the object car has two properties, name and price, and a set price method that will change the price. This set price method will be created every time a new car is created which may or may not be what you want. Creating the set name method every time will increase the overhead of your program. Instead you can add the set price method to the prototype so it will not have to be recreated every time a new car is made. This will make object creation faster and improve the performance of your web page. 
			function car(name) {
				this.name = name;
				this.price = 0;
			}
			
			car.prototype.set_price = function(price) {
				this.price = price;
			}

Public and Private Members

Public data members can be access directly using the dot operator. When you create a data member using the this keyword that makes the data member public. Public data members are specific to each object instance. 
			function Person () {
				this.name = 'null';
			}
			
			var customer = new Person();
			customer.name = 'Larry';
Data members can also be accessed using square brackets and quotes. This is useful when you have key value pairs. 
			function Person () {
				this.name = 'null';
			}
			
			var customer = new Person();

			var key = "name";
			customer[key];
Private data members are only accessible inside of the function or object. Prototyped defined methods also do not have access. To create a private data member simply declare a variable with the var keyword. The scope of the variable will now only be within the object. 
			function myFunction () {
				var privateMember = 10; // private
				this.publicMember = 1; // public
			}

Static Members

Static member variables are shared by all instances of the class. There is only one copy of the static variables that is referenced by each class instance. This is useful to keep track of how many instances there are of a certain class or for mathematical constants. 
			function Car() {
				if(typeof Car.total == 'undefined') {
					Car.total = 0;
				}
				++Car.total;
			}
			
			function Circle(radius) {
				Circle.PI = 3.14;
				this.radius = radius;
			}

Inheritance

Inheritance is when an object is made by coping the properties of another object. JavaScript does not use the classical inheritance model instead uses a prototyped inheritance model. This will allow you to create new objects by coping other objects and adding new data and method. 
		   //parent class
		   function Parent(name) {
				this.name = name;
		   }
		   
		   function Child() {
				this.value = 10;
		   }
		   
		   Child.prototype = new Parent();
		   
		   var instance1 = new Child();
		   
		   //the child class now has access to the name from parent.
